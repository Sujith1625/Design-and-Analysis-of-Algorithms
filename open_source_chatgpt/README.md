# Open Source ChatGPT Project

This project aims to host and serve open source language models through a web-based interface. The structure is inspired by popular conversational AI projects.

## Directory Overview

- `models/` – Downloaded or converted open source models.
- `backend/` – API server and model serving code.
- `frontend/` – Web user interface for interacting with the models.
- `scripts/` – Helper scripts for setup and maintenance.
- `configs/` – Configuration files for deployments and environments.
- `docs/` – Project documentation.

Additional folders and files can be added as the project evolves.
